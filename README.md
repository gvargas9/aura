# Aura

**AI-Native Omni-Commerce Food Platform**

Aura is a next-generation, direct-to-consumer (D2C) and business-to-business (B2B) e-commerce platform built for a unique, all-natural, non-refrigerated food product line.

## Overview

This platform serves as a scalable, multi-tenant "central nervous system" managing:
- B2C subscriptions with "build-a-box" customization
- B2B dealer portals with white-label support
- High-value logistics channels (aviation, marine)
- Unattended retail (smart vending) management

## Tech Stack

| Component | Technology |
|-----------|------------|
| Backend/DB | Supabase (Postgres, Auth, RLS) |
| Automation | n8n |
| Payments | Stripe |
| Communications | Vapi, Twilio |
| AI Hosting | AWS (Bedrock, Lambda) |
| Storage | AWS S3 / Supabase Storage |
